# Introduction to GraphQL

## Recommendations

- Basic knowledge of
  - JavaScript/TypeScript
  - NodeJS
  - React
- NodeJS 8.14.x LTS
- Linux, OSX or Windows operation system
- VS Code
- Chrome and Developer Tools
- Postman, cURL or httpie
