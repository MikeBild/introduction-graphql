# Introduction to GraphQL

## Recommendations

- Basic knowledge of
  - JavaScript
  - NodeJS or Babel-Node (ES6/7)
  - React
- NodeJS 8.10.x LTS
- Linux, OSX or Windows operation system
- Sublime or VS Code
- Chrome and Developer Tools
- Postman, cURL or httpie
